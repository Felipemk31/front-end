Tag input
A tag input é usada para criar controles interativos dentro de um formulário. Ela pode ter vários tipos de entrada, especificados pelo atributo type.

Atributo type
O atributo type da tag input especifica o tipo de entrada que será criado. Alguns valores comuns incluem:

text: Campo de texto simples.
password: Campo de entrada para senhas, onde os caracteres são ocultos.
checkbox: Caixa de seleção, que permite selecionar várias opções.
radio: Botão de opção, que permite selecionar apenas uma opção de um grupo.
submit: Botão de envio do formulário.
date: Campo de entrada para data.
Atributo required
O atributo required é usado para tornar um campo obrigatório, o que significa que o usuário deve preenchê-lo antes de enviar o formulário.

Tag select
A tag select é usada para criar uma lista suspensa (ou um menu dropdown) dentro de um formulário. Ela pode conter uma ou mais opções.

Tipo Radio e CheckBox
Radio: Utilizado para criar um botão de opção onde apenas uma opção pode ser selecionada de um grupo de opções.
CheckBox: Utilizado para criar caixas de seleção onde várias opções podem ser selecionadas.

Tipo Submit
Submit: Cria um botão que, quando clicado, envia o formulário para o servidor.

Tipo Date Date: Cria um campo de entrada para inserir uma data.
Tipo Password Password: Cria um campo de entrada onde os caracteres inseridos são ocultos.
