Como alinhar imagens e textos com divs e CSS

Utiliza-se da tag div(utilizar o seletor display com o valor flex no CSS)
para criar uma area em que é possivel alinhar objetos dentro dela.
Utiliza-se do seletor box-aling com os valores:
Para alinhar a imagem à esquerda, use a palavra left;
Já para alinhar a imagem à direita, use a palavra right;
No caso de imagens no meio, ou seja, centralizadas, use a palavra middle;
No topo, use a palavra top;
Embaixo, use a palavra bottom.
para colocar imagens alinhadas uma
embaixo da outra basta criar outra div
abaixo com a outra imagem.
